# Dagobah Fridge Pickles — Recipe Scaler

A single-page recipe scaler for fridge pickles. Adjust your container size and the ingredient quantities update automatically.

## Features

- Scale ingredients by container size (liters, quarts, or gallons)
- Slider and manual input for precise sizing
- Quick presets for common container sizes (500 ml, 1 qt, 1.3 L, 1/2 gal, 1 gal)
- Ingredient checkboxes to track what you've added
- Automatic unit upgrades (tsp to tbsp when quantities are large enough)
- Amounts rounded to nearest 1/8 tsp/tbsp for practicality

## Ingredients (base recipe — 1.3 L)

- White Vinegar
- Salt
- Calcium Chloride
- Black Peppercorns
- Mustard Seed
- Red Pepper Flakes
- Dill Weed
- Garlic Powder

## Usage

Open `index.html` in a browser. No build step or dependencies required.
